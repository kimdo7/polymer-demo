### Getting Started with Polymer.js

This project is a collection of demo files for the course on Pluralsight
called "Getting Started with Polymer.js"

### Installation Steps

1. Run git bash or terminal and navigate to the "src" folder
2. Install Node
3. Run "npm isntall -g bower"
4. Run "npm isntall -g polyserve"
5. Run polyserve
6. You can find at http://localhost:8080/components/domes/src
